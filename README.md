Data Analysis Portfolio: Wrangling, EDA & Visualization

This repository showcases a comprehensive data analysis workflow.

Structured Labs: Covers the end-to-end data pipeline, including Data Wrangling (cleaning/normalization), Exploratory Data Analysis (statistical summaries), and Data Visualization (dashboarding).

Applied Project: A deep-dive analysis of an Insurance dataset (EDA 1 insurance data .ipynb), utilizing boxplots for outlier detection and correlation matrices to identify key drivers of insurance charges (e.g., smoker status and BMI).
